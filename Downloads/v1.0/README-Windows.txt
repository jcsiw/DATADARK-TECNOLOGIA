DATADARK REMOTE SUPPORT - WINDOWS

Versão:
1.0

Base:
RustDesk 1.4.7

Servidor:
suporte.datadark.com.br

Instalação:

1. Execute DATADARK-RemoteSupport-Setup.exe como Administrador.
2. Aguarde a conclusão da instalação.
3. Abra o DATADARK Remote Support.
4. O sistema será configurado automaticamente para utilização do servidor DATADARK.

Recursos:

* Servidor próprio DATADARK
* Relay próprio DATADARK
* Chave pública validada
* Atualizações automáticas desativadas
* Compatível com Windows 10 e Windows 11
