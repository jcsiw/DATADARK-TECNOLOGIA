DATADARK REMOTE SUPPORT - LINUX

Versão:
1.0

Base:
RustDesk 1.4.7

Servidor:
suporte.datadark.com.br

Instalação:

1. Abra um terminal.
2. Execute:

sudo chmod +x DATADARK-RemoteSupport-Linux.run
sudo ./DATADARK-RemoteSupport-Linux.run

3. Aguarde a conclusão.
4. Execute RustDesk normalmente.

Recursos:

* Servidor próprio DATADARK
* Relay próprio DATADARK
* Chave pública validada
* Atualizações automáticas desativadas
* Compatível com Debian, Ubuntu, Linux Mint, MiniOS e derivados
